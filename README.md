## douban.fm
* 简洁的豆瓣电台命令行版，基于 Node.js。

## 截屏
![image](https://raw.githubusercontent.com/shuiszhang/douban.fm/master/douban.fm.png)

## 功能特性
* 支持登录
* 支持同步收听记录
* 支持播放红心兆赫
* 支持标记红心/取消红心/丢垃圾桶
* 支持歌词显示
* 支持暂停/恢复播放

## 依赖
* Node.js 7.6+
* mplayer

## 安装
* npm i doubanfm -g

## 命令行启动
* doubanfm

## 快捷键
* j 光标下移，选择兆赫
* k 光标上移，选择兆赫
* enter/return(回车) 切换到当前选择兆赫
* n 播放下一首
* r 标记红心
* u 取消红心
* b 丢垃圾桶
* space(空格) 播放/暂停
